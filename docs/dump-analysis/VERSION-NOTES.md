# Dump Analysis Version Notes

The artifacts in this directory were collected across multiple pipeline versions and dist builds. Some files reflect older code states.

## Files without Tier:/Pipeline:

- `pi-dump.scout.1778860570616.md` — generated by an older dist build before the `Tier:` and `Pipeline:` header fields were added.
- These files are valid historical artifacts but should not be used as a reference for the current dump format.

## Files with Tier:/Pipeline:

- `pi-dump.build.1778872299999.md` and others with `Tier:` / `Pipeline:` were generated by the current dist build and reflect the canonical format.

## Current format expectations

Dumps produced by the current code include:
- HTML comment header with `Passes:`, `Flow:`, `Tier:`, `Pipeline:`, `Generated:`
- `.txt` twin file alongside every `.md` dump
- Compression stats block with `passesApplied` (the `pipelineVersion` is present in the HTML header, not the JSONL entry)

When comparing or analyzing artifacts, always check whether the file matches the current format before treating it as a ground-truth reference.
